This project was a small test to try and play around with the technology behind natural language processing using the tensor flow library to drive it. This project was a section I created for a Covid Support mobile web app. Its purpose was to display to the user a “Heat map” of Glasgow’s neighborhoods, were based on how many cases of covid there were recorded in that area the colour of the neighbourhood would change. Unfortunately there were no open source statistics directly related to the number of covid cases found in a neighbourhood so ALL data presented in the app is random. The app also allows the user to sort based on population and vaccinations. All files again can be found on my github.
